## CI/CD 流水线

```
开发者 git push
       │
       ▼
GitHub Actions
  ├── lint + typecheck
  ├── Vitest 单元测试
  ├── Playwright E2E 测试
  └── ✅ 全绿 → Vercel 自动部署
              └── Sentry 记录新版本
```

## 关键设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 框架 | Next.js 15 App Router | RSC 减少客户端 JS，SEO 友好 |
| 数据库 | Supabase + Prisma | 免运维 + 类型安全 ORM |
| 认证 | Supabase Auth | 免费额度大，Magic Link 开箱即用 |
| 样式 | Tailwind + shadcn/ui | 高速 UI 搭建，组件所有权在项目中 |
| 部署 | Vercel | Push 即上线，免运维 |
| 监控 | Sentry | 错误直达 VS Code 行号 |
| 计费 | Stripe | 全球标准，Webhook 成熟 |

## 本地快速启动

```bash
# 1. 安装依赖
npm install

# 2. 复制并填写环境变量
cp .env.example .env.local

# 3. 推送数据库 schema
npx prisma db push

# 4. 启动开发服务器
npm run dev

# 5. 运行 E2E 测试（需先启动 dev server）
npm run test:e2e
```
